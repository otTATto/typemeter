## ⌨️ What is typemeter?

typemeter is a local activity meter for your typing — like a pedometer, but for your keyboard.
It counts your keystrokes without ever recording what you type, and all data stays on your device.

## ✨ New Features

- **Background residency**: Closing the window no longer quits the app — typemeter keeps counting in the background. Control it from the system tray (Windows) or menu bar (macOS): Open / Settings… / Quit (#18)
- **Auto launch at PC login**: typemeter now starts automatically when you log in (enabled by default, can be turned off in Settings) and launches silently in the tray without opening a window (#34)
- **Settings window**: Added a Settings window (menu > Settings…) with Light/Dark Theme, Always on Top, and Auto Launch at PC Login toggles (#28)
- **Always on Top**: Pin the main window above other apps, from the header toggle or the Settings window (#28)
- **New app icon**: Applied the new keycap logo across all platforms (#29)

## 🐛 Bug Fixes

- Fixed a brief flash of the light theme when opening a window in dark mode
- Launching the app while it is already running now brings up the existing window instead of starting a second process

> [!NOTE]
> 🍎 **macOS users**:
>
> - The app is not code-signed. After moving it to the Applications folder, open Terminal and run the following command:
>   ```sh
>   sudo xattr -d com.apple.quarantine /Applications/typemeter.app
>   ```
> - On first launch, Gatekeeper may block it.
> - Go to System Settings → Privacy & Security → Input Monitoring, then turn on the toggle for typemeter.app.
>
> 🪟 **Windows users**:
>
> - If SmartScreen shows a warning, click "More info" → "Run anyway".
